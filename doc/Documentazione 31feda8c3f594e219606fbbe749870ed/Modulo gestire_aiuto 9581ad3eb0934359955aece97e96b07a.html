Funzione gestire_menu_aiuto

Gestisce il menu di aiuto.

INPUT

NomeDescrizioneTipo/vincoli
sceltarappresenta l’azione che l’utente vuole effettuare
0: Esci
1: Stampare le regole del gioco
2:Stampare il manuale
SCELTA_USCIRE_DAL_MENU ≤ Intero ≤ SCELTA_STAMPARE_MANUALE
NOME_FILE_REGOLE_GIOCOnome del file regole_giocoarray a una dimensione di caratteri
NOME_FILE_MANUALE_GIOCOnome del file manuale_giocoarray a una dimensione di caratteri
NOME_FILE_MENU_AIUTOnome del file menu_aiutoarray a una dimensione di caratteri
MESSAGGIO_SCELTA_AZIONE_MENUindica il messaggio generico da stampare al momento dell’inserimento di una scelta all’interno di un menuarray a una dimensione di caratteri
SCELTA_USCIRE_DAL_MENUvalore utilizzato per uscire dal menùintero
PRIMA_COORDINATA_SCHERMOvalore che corrisponde prima coordinata dello schermointero
SCELTA_STAMPARE_MANUALEindica il valore da inserire per la visualizzazione del manualeintero

ALGORITMO

INIZIO

ESEGUI

// Sposta il cursore alla prima coordinata dello schermo

spostare_cursore(PRIMA_COORDINATA_SCHERMO, PRIMA_COORDINATA_SCHERMO)

// Stampare il menù aiuto

stampare_file_di_testo(NOME_FILE_MENU_AIUTO)

// Richiedere un valore intero per scegliere l'azione del menù e ne attende l'inserimento

riga = ottenere_riga_cursore()

scelta = chiedere_intero(MESSAGGIO_SCELTA, SCELTA_USCIRE_DAL_MENU, SCELTA_STAMPARE_MANUALE, riga, PRIMA_COORDINATA_SCHERMO)

// Gestisce la scelta dell'utente
gestire_scelta_menu_aiuto(scelta, NOME_FILE_REGOLE_GIOCO, NOME_FILE_MANUALE_GIOCO)

FINCHE (scelta ≠ SCELTA_USCIRE_DAL_MENU)

FINE

FINE
