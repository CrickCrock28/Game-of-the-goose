RECORD record_menu_partite_salvate, contiene il file delle partite salvate e eventualmente una partita cancellata o letta dall’elenco:

  • partite_salvate, elenco partite salvate dall’utente fino a questo momento, file binario(contiene come primo dato il numero di partite salvate (da 0 a 5), e successivamente quante volte il numero precedente di partite salvate come record_partita)
  • partita_opzionale, partita cancellata dall’elenco o letta dall’elenco elenco, record_partita
