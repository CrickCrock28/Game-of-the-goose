RECORD record_partita, contiene tutti i dati relativi ad una partita:

  • percorso, record che contiene il percorso, record_percorso
  • vet_giocatori, record che contiene tutti i giocatori, record_vet_giocatori
  • indice_giocatore_di_turno, NUMERO_MINIMO_GIOCATORI≤intero≤campo dimensione di vet_giocatori
  • ultimo_lancio_dado_1, ultimo lancio del primo dado effettuato dal giocatore, NUMERO_MINIMO_DADO≤intero≤NUMERO_MASSIMO_DADO
  • ultimo_lancio_dado_2, ultimo lancio del secondo dado effettuato dal giocatore, NUMERO_MINIMO_DADO≤intero≤NUMERO_MASSIMO_DADO
  • nuova_partita, indica se l’utente vuole creare una nuova partita e di quella in corso, booleano
  • abbandona_partita, indica se l’utente vuole abbandonare la partita corrente, booleano
  • terminata, indica se la partita è terminata, booleano
  • salvare_partita, indica se l’utente vuole salvare la partita, booleano
Funzione copiare_record_partita

Copia i dati di una partita in un’altra.

INPUT

partita_in_cui_copiare, partita in cui copiare, record_partita

partita_da_copiare, partita da copiare, record_partita

OUTPUT

partita_in_cui_copiare, partita con dati copiati, record_partita

ALGORITMO

INIZIO

campo percorso di partita_in_cui_copiare= copiare_record_percorso(campo percorso di partita_in_cui_copiare, campo percorso di partita_da_copiare)

campo vet_giocatori di partita_in_cui_copiare= copiare_record_vet_giocatori(campo vet_giocatori di partita_in_cui_copiare, campo vet_giocatori di partita_da_copiare)

campo indice_giocatore_di_turno di partita_in_cui_copiare= campo indice_giocatore_di_turno di partita_da_copiare

campo terminata di partita_in_cui_copiare= campo terminata di partita_da_copiare

campo ultimo_lancio_dado_1 di partita_in_cui_copiare= campo ultimo_lancio_dado_1 di partita_da_copiare

campo ultimo_lancio_dado_2 di partita_in_cui_copiare= campo ultimo_lancio_dado_2 di partita_da_copiare

campo nuova_partita di partita_in_cui_copiare= campo nuova_partita di partita_da_copiare

campo abbandona_partita di partita_in_cui_copiare= campo abbandona_partita di partita_da_copiare

FINE
