  • dimensione, dimensione dell'array giocatori, NUMERO_MINIMO_GIOCATORI≤intero≤NUMERO_MASSIMO_GIOCATORI

Funzione leggere_dimensione_record_vet_giocatori

Legge la dimensione del vettore.

INPUT
vet, vettore di cui leggere la dimensione, record_vet_giocatori

OUTPUT
dim, numero di giocatori di vet, NUMERO_GIOCATORI_MINIMO≤intero≤NUMERO_GIOCATORI_MASSIMO

ALGORITMO
INIZIO

dim = elemento del campo dimensione di vet

FINE

OUTPUT
vet, vettore con dimensione aggiornata, vet

ALGORITMO
INIZIO

elemento del campo dimensione di vet = dim

Funzione scrivere_giocatore_record_vet_giocatori

INPUT
vet, vettore di cui scrivere il nuovo giocatore, record_vet_giocatori

giocatore, giocatore da scrivere in vet, record_giocatore

vet, vettore con scritto il nuovo giocatore, record_vet_giocatori

Funzione copiare_record_vet_giocatori

Copia i dati del secondo vettore di giocatori dato in input all’interno del primo vettore di giocatori dato in input.

INPUT

vet_1, vettore di giocatori in cui copiare i dati di vet_2, record_vet_giocatori

vet_2, vettore di giocatori da cui copiare i dati in vet_1, record_vet_giocatori

OUTPUT

vet_1, vettore di giocatori con i dati di vet_2 copiati, record_vet_giocatori

LAVORO

i, indice dei vettore di giocatori, NUMERO_MINIMO_GIOCATORI≤intero≤NUMERO_MASSIMO_GIOCATORI

ALGORITMO

INIZIO

vet_1= scrivere_dimensione_record_vet_giocatori(vet_1, leggere_dimensione_record_vet_giocatori(vet_2))

i = PRIMO_INDICE_ARRAY

MENTRE (i ≤ NUMERO_MASSIMO_CASELLE)

vet_1 = scrivere_giocatore_record_vet_giocatori(vet_1, i, leggere_giocatore_record_vet_giocatori(vet_2, i))

i = i + 1

FINE

FINE
