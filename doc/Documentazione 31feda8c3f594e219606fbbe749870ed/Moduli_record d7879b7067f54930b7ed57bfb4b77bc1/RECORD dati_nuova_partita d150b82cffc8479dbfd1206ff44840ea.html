RECORD record_dati_nuova_partita, contiene i dati necessari alla creazione di una nuova partita:

  • numero_giocatori, numero di giocatori della nuova partita, NUMERO_MINIMO_GIOCATORI≤intero≤NUMERO_MASSIMO_GIOCATORI
  • numero_caselle, numero di caselle della nuova partita, NUMERO_MINIMO_CASELLE≤intero≤NUMERO_MASSIMO_CASELLE
Funzione leggere_numero_giocatori_dati_nuova_partita

Legge il numero di giocatori della nuova partita.

INPUT
dati_nuova_partita, dati della nuova partita, record_dati_nuova_partita

OUTPUT
numero_giocatori; numero di giocatori della nuova partita, NUMERO_MINIMO_GIOCATORI≤intero≤NUMERO_MASSIMO_GIOCATORI

ALGORITMO
INIZIO

numero_giocatori= elemento del campo numero_giocatoridi dati_nuova_partita

FINE

Funzione scrivere_numero_giocatori_dati_nuova_partita

Scrive il numero di giocatori della nuova partita.

INPUT
dati_nuova_partita, dati della nuova partita, record_dati_nuova_partita
numero_giocatori: nuovo numero di giocatori della nuova partita, NUMERO_MINIMO_GIOCATORI≤intero≤NUMERO_MASSIMO_GIOCATORI

OUTPUT
dati_nuova_partita, dati della nuova partita con il nuovo numero di giocatori, record_dati_nuova_partita

ALGORITMO
INIZIO

elemento del campo numero_giocatori di dati_nuova_partita = numero_giocatori

FINE
