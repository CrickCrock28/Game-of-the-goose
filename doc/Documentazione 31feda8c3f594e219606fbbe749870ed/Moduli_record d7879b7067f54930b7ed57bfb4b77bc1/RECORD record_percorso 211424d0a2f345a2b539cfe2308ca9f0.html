RECORD record_percorso, contiene i dati del percorso:

  • dimensione, numero di caselle del percorso, NUMERO_MINIMO_CASELLE<=intero<=NUMERO_MASSIMO_CASELLE
  • caselle, insieme delle caselle del percorso, array a 1 dimensione di caratteri, di lunghezza NUMERO_MASSIMO_CASELLE
Funzione leggere_dimensione_record_percorso

Legge la dimensione del percorso.

INPUT
percorso, percorso di cui leggere la dimensione, record_percorso

OUTPUT
dimensione, dimensione del percorso, NUMERO_MINIMO_CASELLE≤intero≤NUMERO_MASSIMO_CASELLE

ALGORITMO
INIZIO

dimensione = campo dimensione di percorso

FINE

Funzione scrivere_dimensione_record_percorso

Scrive la dimensione del percorso.

INPUT
percorso, percorso di cui scrivere la dimensione, record_percorso
dimensione, nuova dimensione del percorso, NUMERO_MINIMO_CASELLE≤intero≤NUMERO_MASSIMO_CASELLE

OUTPUT
percorso, percorso con la nuova dimensione, record_percorso

ALGORITMO
INIZIO

campo dimensione di percorso = dimensione

FINE

Funzione leggere_casella_record_percorso

Legge la casella del percorso.

INPUT
percorso, percorso di cui leggere la casella, record_percorso
posizione, posizione della casella all'interno del percorso, PRIMO_INDICE_ARRAY ≤ intero ≤ dimensione del percorso

OUTPUT
casella, casella del percorso alla posizione indicata, carattere, deve essere uguale a uno di questi caratteri TIPO_CASELLA_OCA, TIPO_CASELLA_PONTE, TIPO_CASELLA_LOCANDA, TIPO_CASELLA_POZZO, TIPO_CASELLA_LABIRINTO, TIPO_CASELLA_SCHELETRO, TIPO_CASELLA_PRIGIONE e TIPO_CASELLA_FINE

ALGORITMO
INIZIO

casella = campo caselle in posizione posizione di percorso

FINE

Funzione scrivere_casella_record_percorso

Scrive la casella del percorso.

INPUT
percorso, percorso di cui scrivere la casella, record_percorso
posizione, posizione della casella all'interno del percorso, PRIMO_INDICE_ARRAY ≤ intero ≤ dimensione del percorso
casella, nuova casella da scrivere nella posizione indicata, carattere, deve essere uguale a uno di questi caratteri TIPO_CASELLA_OCA, TIPO_CASELLA_PONTE, TIPO_CASELLA_LOCANDA, TIPO_CASELLA_POZZO, TIPO_CASELLA_LABIRINTO, TIPO_CASELLA_SCHELETRO, TIPO_CASELLA_PRIGIONE e TIPO_CASELLA_FINE

OUTPUT
percorso, percorso con la casella scritta nella posizione indicata, record_percorso

ALGORITMO
INIZIO

campo caselle in posizione posizione di percorso = casella

FINE

Funzione copiare_record_percorso

Copia i dati del secondo percorso dato in input all’interno del primo percorso dato in input.

INPUT

percorso_1, percorso in cui copiare i dati di percorso_2, record_percorso

percorso_2, percorso da cui copiare i dati in percorso_1, record_percorso

OUTPUT

percorso_1, percorso con i dati di percorso_2 copiati, record_percorso

LAVORO

i, indice di percorso, NUMERO_MINIMO_CASELLE≤intero≤NUMERO_MASSIMO_CASELLE

ALGORITMO

INIZIO

percorso_1 = scrivere_dimensione_record_percorso(percorso_1, leggere_dimensione_record_percorso(percorso_2))

i = PRIMO_INDICE_STRINGA

MENTRE (i ≤ NUMERO_MASSIMO_CASELLE)

percorso_1 = scrivere_casella_record_percorso(percorso_1, i, leggere_casella_record_percorso(percorso_2, i))

i = i + 1

FINE

FINE
